# AI-TRPG Collaboration Platform

An AI-assisted collaboration platform for TRPG game masters and players, focused on campaign management, character sheets, session logs, and AI-generated recaps.

## Product Background

TRPG game masters often need to manage world settings, character sheets, NPCs, clues, session logs, and post-session recaps across multiple tools such as documents, chat apps, spreadsheets, and dice bots. This project aims to provide a lightweight web-based workspace that centralizes campaign information and uses AI to reduce the cost of session recap and preparation.

## Core Users

- Game Masters / Keepers / Dungeon Masters
- TRPG players participating in long-term campaigns

## MVP Features

- User authentication
- Campaign management
- Member roles and permissions
- Character sheet management
- Session log management
- AI-generated session summaries
- AI-generated NPC drafts

## Tech Stack
